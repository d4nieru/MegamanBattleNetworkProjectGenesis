# About this project
*(The following description is taken from their gamejolt page)*

MMBN Project Genesis is a fan game of the MegaMan Battle Network games. Made by people who didn't like the ending of 6 and thought the game should continue after CX, the game takes place 2 years after 6, and follows the mysteries of MegaMan, Omega, and over a dozen new characters. (Megaman Battle Network Chrono X).
